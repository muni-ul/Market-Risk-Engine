# Summary Report Excerpt

This is a representative excerpt from `results/demo_run/summary_report.md`. The real values are generated by the CLI run.

```markdown
# PyRiskLab Run Summary

Run name: `demo_run`
Seed: `42`
Output directory: `results/demo_run`

## Simulation Only

This is a local simulation only. It does not use live market data, place real trades, connect to a brokerage, provide investment advice, or make profitability claims.

## Market Simulation

- Initial underlying price: $100.00
- Final underlying price: generated from the seeded synthetic path
- Generated steps: controlled by `configs/demo.yaml`
- Model: geometric Brownian motion with synthetic data
- Drift assumption: from config
- Volatility assumption: from config

## Option Contract

- Symbol: configured option symbol
- Type: call or put
- Strike: configured strike
- Initial days to expiry: configured expiry horizon
- Initial model price: generated by Black-Scholes
- Final model price: generated by Black-Scholes

## Strategy Signals

- Total signal rows: generated by the strategy module
- Proposed simulated orders: generated by the order builder
- Approved simulated orders: generated by risk validation
- Blocked simulated orders: generated by risk validation
- Skipped simulated orders: generated when fake execution is disabled

## Portfolio Results

- Starting cash: configured risk starting cash
- Number of simulated trades: generated by fake execution
- Final portfolio value: generated from portfolio accounting
- Max drawdown: generated from portfolio history
- Risk events: generated by the risk manager

## Fake Execution

Generated from fake execution state, or reports that fake execution was disabled.

## Risk Events

Generated from risk manager state; empty state says no risk events were triggered.

## Benchmark

Vectorized NumPy pricing ran `<generated speedup>`x faster than the Python loop on this machine. Benchmark results vary by hardware, Python version, and input size.
```
