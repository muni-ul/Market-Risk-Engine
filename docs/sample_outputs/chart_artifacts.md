# Chart Artifact Reference

The demo run writes static PNG charts under `results/<run_name>/`. These charts are generated locally with matplotlib and are intended for quick reviewer inspection, README screenshots, and interview walkthroughs.

## Generated Charts

| File | Purpose | Expected content |
| --- | --- | --- |
| `market_path.png` | Market simulation preview | Simulated underlying price by step |
| `option_price.png` | Pricing result preview | Black-Scholes option price by step |
| `greeks.png` | Sensitivity preview | Delta, gamma, vega, theta, and rho lines |
| `portfolio_value.png` | Portfolio state preview | Simulated total portfolio value by step |
| `drawdown.png` | Risk/accounting preview | Drawdown percentage by step |

## Reviewer Notes

- The charts are generated by running `python -m pyrisklab run --config configs/demo.yaml --overwrite`.
- The repository does not commit generated `results/demo_run/` PNGs because chart metadata and benchmark timing can vary by machine.
- Every chart should have a readable title, labeled axes, and a deterministic data source from the seeded config.
- These charts are simulation artifacts only; they are not live market data, trading recommendations, or profitability evidence.
