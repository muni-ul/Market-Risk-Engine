# Chart Artifact Reference

The demo run writes static PNG charts under `results/<run_name>/`. These charts
are generated locally with matplotlib and are intended for quick reviewer
inspection, README screenshots, and interview walkthroughs.

## Generated Charts

- `market_path.png`: market simulation preview with simulated underlying price
  by step.
- `option_price.png`: pricing result preview with Black-Scholes option price by
  step.
- `greeks.png`: sensitivity preview with delta, gamma, vega, theta, and rho
  lines.
- `portfolio_value.png`: portfolio state preview with simulated total portfolio
  value by step.
- `drawdown.png`: risk/accounting preview with drawdown percentage by step.

## Reviewer Notes

- The charts are generated by running
  `python -m pyrisklab run --config configs/demo.yaml --overwrite`.
- The repository does not commit generated `results/demo_run/` PNGs because
  chart metadata and benchmark timing can vary by machine.
- Every chart should have a readable title, labeled axes, and a deterministic
  data source from the seeded config.
- These charts are simulation artifacts only; they are not live market data,
  trading recommendations, or profitability evidence.
